# 儿童趣味打字练习 \- README

一个专为儿童设计的趣味打字练习网页应用，支持中英文双语练习，带有成就系统和语音朗读功能。纯前端实现，无需后端，可直接在浏览器运行或部署到 GitHub Pages。

---

## ✨ 功能特性

### 两大练习模式

- **全文练习模式**：单栏全屏，完整保留中英内容，全文打字练习

- **双语对照模式**：左右分栏，中文对照显示，仅练习英文部分

### 辅助学习功能

- 📊 **实时统计**：打字速度、准确率、用时、全文完成进度

- 🎯 **逐字符高亮**：正确绿色、错误红色、当前光标闪烁提示

- 🔊 **语音朗读**：单词朗读、全文朗读，支持调节语速和音量

- 🏆 **成就系统**：连击贴纸解锁、星星动画、完成鼓励文案

- 🎨 **主题切换**：日间 / 暗色双主题，保护视力

- 🔤 **字号调节**：自由调节练习文本字号，适配不同视力

- ⌨️ **虚拟键盘**（电脑端）：指法引导、按键高亮、正确 / 错误反馈

- 🎤 **语音选择**：自动优选响亮语音 \+ 手动选择多种语音包

- 📚 **内置题库**：五年级上下册，广州版教科版英语教材，单词 / 短语 / 句型 / 课文四类练习

- 🔊 **音效开关**：可控制按键音、错误音、完成音的开关，本地持久化保存

- 📱 **移动端适配**：完美支持手机 / 平板触屏打字

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 下载或克隆本项目到本地

2. 双击 `index.html` 即可在浏览器中打开使用

### 方式二：GitHub Pages 部署

1. 将项目上传到 GitHub 仓库

2. 进入仓库 Settings → Pages

3. 选择 main 分支作为源，保存

4. 访问生成的网址即可在线使用

## 📝 使用说明

### 基本使用流程

1. 在文本框中粘贴练习内容，或点击「上传 TXT 文件」导入，或点击「选择题库」选择内置练习

2. 点击「开始练习」按钮

3. 在输入框中打字，系统会实时统计和反馈

4. 完成全部内容后自动弹出成绩报告

### 练习模式选择

- 检测到文本包含中文时，会弹出模式选择弹窗

- **「确定」**：全文练习模式（单栏全屏，完整保留中英内容）

- **「取消」**：双语对照模式（左右分栏，中文对照，仅练习英文）

### 内置题库使用

1. 点击「📚 选择题库」按钮

2. 选择上册或下册

3. 展开模块，选择单元

4. 点击练习类型（单词 / 短语 / 句型 / 课文）自动填充

### 使用小贴士

- 带音标的英文文本会自动删除 `[xxx]` 音标内容

- 所有配置自动本地保存，下次打开无需重新设置

- 练习最后一行，输入完成后自动弹出成绩

- 手机端建议使用系统自带浏览器，语音朗读更稳定

- 电脑端可开启虚拟键盘进行指法练习

- 音效可在「更多设置」中开关
