## 📝 v2\.6\.2 更新日志

### 🐛 问题修复

1. **修复陨石雨语法致命错误**：补齐 `spawnMeteorRain` 函数 for 循环闭合大括号，修正护盾消耗分支多余 `if` 包裹赋值语句的语法问题，解决陨石雨事件完全无法触发的故障。

### ✨ 功能优化

1. **陨石雨范围优化**：限制陨石横向生成区间为 25%\~75%，陨石仅在玩家与怪物中间的战斗区域生成，避免掉落至角色外侧空白区域，提升视觉合理性。

2. **移动端角色选择 UI 自适应重构**：角色卡片全元素采用 `clamp` 流体适配方案，头像字号、卡片内边距、网格间距、文字大小随屏幕宽度平滑缩放，解决窄屏手机下角色头像过大、排版拥挤、内容挤压的问题。

3. **多端视觉统一**：PC 端（宽度≥641px）自动锁定原有固定尺寸，保持设计规范；移动端自动适配缩放，兼顾大屏与小屏体验。

## 🎮 核心功能

### 基础打字练习

- 支持自定义文本粘贴、TXT 文件上传、内置教材题库三种练习内容来源

- 实时统计打字速度、准确率、全文进度，配套进度条可视化

- 单词朗读、全文朗读功能，支持 0\.25x\~1\.5x 多档语速调节

- 昼夜主题切换、字号无级调节，适配不同使用习惯

- 内置虚拟键盘，适配触屏设备使用

### 战斗模式

- 6 种可选玩家角色：平衡型、攻击型、高攻型、暴击型、肉盾型、回复型，差异化属性与被动技能

- 6 种梯度怪物对手，4 档难度，自适应 AI 动态调节攻击节奏

- 完整战斗系统：血量、护盾、怒气大招、受击 / 攻击动画、伤害飘字特效

- 打断机制：怪物攻击预警期内，连续输入 3 个正确字母可打断攻击

- 结算统计面板，支持再来一局、返回重选角色

### 场景互动随机事件

- **治疗星星⭐**：完整输入当前句子即可拾取，回复 20% 最大生命值

- **陨石雨☄️**：天罚机制，双方同时扣除 6% 最大生命值，支持护盾、无敌状态抵挡

- **能量球⚡**：正确输入目标字母即可拾取，获得 3 个句子的伤害翻倍增益

- 事件定时随机触发，增加战斗随机性与趣味性

## 🚀 使用方法

### 本地运行

1. 下载全部文件至本地目录

2. 直接使用浏览器打开 `index.html` 即可启动使用

3. 推荐使用 Chrome、Edge、手机系统自带浏览器，获得最佳语音朗读体验

### 开启战斗模式

1. 先选择 / 粘贴练习内容，确保有效句子数量≥3

2. 点击主页「⚔️ 战斗模式」按钮进入角色选择页

3. 分别选择玩家角色与对手怪物，点击「开始战斗」即可
