# 儿童趣味打字练习 \- README

一个专为儿童设计的趣味打字练习网页应用，支持中英文双语练习，带有成就系统和语音朗读功能。纯前端实现，无需后端，可直接在浏览器运行或部署到 GitHub Pages。

---

## ✨ 功能特性

### 🎯 两大练习模式

- **全文练习模式**：单栏全屏，完整保留中英内容，全文打字练习

- **双语对照模式**：左右分栏，中文对照显示，仅练习英文部分

### 📊 实时统计反馈

- **打字速度**：实时计算每分钟输入单词数（WPM）

- **输入准确率**：双指标统计，准确率 \+ 完成度分开显示

- **练习用时**：精确到秒的计时功能

- **完成进度**：实时显示全文完成百分比

- **进度条可视化**：速度、准确率、进度三条进度条直观展示

### 🔤 逐字符高亮校验

- **正确字符**：绿色高亮 \+ 弹跳动画反馈

- **错误字符**：红色高亮提示

- **当前光标**：闪烁提示，清晰定位

- **待输入字符**：灰色显示，一目了然

### 🎤 双引擎语音系统

**在线语音引擎（推荐）**

- 基于 Cloudflare Worker \+ 百度翻译 TTS 代理

- 解决移动端系统语音加载不出来的问题

- 音质清晰稳定，不受系统语音包限制

- 自定义域名部署，跨域访问正常

**系统语音引擎**

- 本地语音合成，无需网络

- 支持多种系统语音包选择

- 自动优选响亮清晰的语音

**语音功能**

- 🔊 全文朗读：逐句播放，高亮同步滚动

- 🔤 单词朗读：单词输入完成自动朗读

- 📝 行朗读：整行输入完成自动朗读

- ⚡ 语速调节：0\.25x \- 1\.5x 七档语速

- 🔉 音量调节：轻柔 / 标准 / 响亮三档

### 🏆 成就激励系统

- **连击贴纸**：5/10/15 连击解锁成就贴纸

- **星星动画**：连击时飘星特效，增加趣味性

- **连击提示**：实时显示连击数和鼓励文字

- **完成弹窗**：练习完成后详细成绩报告

- **鼓励文案**：根据成绩智能生成鼓励语

### 🎨 个性化设置

- **主题切换**：日间 / 暗色双主题，保护视力

- **字号调节**：自由调节练习文本字号，适配不同视力

- **音效开关**：按键音、错误音、完成音可独立控制

- **单词朗读开关**：可控制是否自动朗读输入的单词

- **本地持久化**：所有设置自动保存，刷新不丢失

### ⌨️ 虚拟键盘（电脑端）

- **指法引导**：下一个按键自动高亮提示

- **按键反馈**：正确绿色、错误红色，即时反馈

- **大小写切换**：自动同步 Shift 状态

- **数字符号键盘**：自动切换数字符号布局

- **手机端自动隐藏**：移动端不显示虚拟键盘

### 📚 内置题库系统

**教材版本**

- 广州版（教科版）英语教材，三年级起点

**覆盖范围**

- 五年级上册 \+ 下册，共 12 个模块，24 个单元

**练习类型**

- 🔤 单元单词：每单元核心词汇

- 📝 重点短语：常用短语搭配

- 💬 重点句型：经典句型练习

- 📖 课文对话：完整课文中英对照

**角色人名**

- 保留教材原有人名：Janet、Jiamin、Ben、Xiaoling、Mike 等

- 角色名前缀自动跳过，不参与输入统计

### 📱 移动端深度适配

- 触屏打字优化，滚动流畅不卡顿

- 软键盘弹出自动适配，防止遮挡

- 智能滚动定位，光标始终在可视区域

- 在线语音支持移动端 playsinline，不会全屏播放

- OPPO 等安卓手机兼容性优化

## 📝 使用说明

### 基本使用流程

1. **选择练习内容**

    - 在文本框中粘贴练习内容

    - 或点击「上传 TXT 文件」导入

    - 或点击「📚 选择题库」选择内置练习

2. **开始练习**

    - 点击「开始练习」按钮

    - 检测到中文会弹出模式选择弹窗

3. **打字练习**

    - 在输入框中打字，系统实时统计和反馈

    - 正确字符变绿，错误字符变红

    - 连续正确触发连击特效

4. **完成练习**

    - 输完最后一行自动弹出成绩报告

    - 查看详细数据：用时、完成度、准确率等

    - 可选择重新练习或清空文本

### 练习模式选择

检测到文本包含中文时，会弹出模式选择弹窗：

- **「确定」** → 全文练习模式（单栏全屏，完整保留中英内容）

- **「取消」** → 双语对照模式（左右分栏，中文对照，仅练习英文）

### 内置题库使用

1. 点击「📚 选择题库」按钮

2. 选择上册或下册 Tab

3. 点击模块展开 / 收起（手风琴效果）

4. 选择具体单元

5. 点击练习类型（单词 / 短语 / 句型 / 课文）自动填充

### 语音设置

1. 点击「更多设置」展开设置面板

2. **语音选择**：在线女声（推荐）或系统语音

3. **语速调节**：拖动滑块调整朗读速度

4. **音量调节**：点击音量按钮切换档位（轻柔🔈 / 标准🔉 / 响亮🔊）

### 使用小贴士

- 带音标的英文文本会自动删除 `[xxx]` 音标内容

- 所有配置自动本地保存，下次打开无需重新设置

- 练习最后一行，输入完成后自动弹出成绩

- 手机端建议使用系统自带浏览器，语音朗读更稳定

- 电脑端可开启虚拟键盘进行指法练习

- 音效可在「更多设置」中开关

- 课文中的角色人名会自动跳过，不影响打字统计
