# 儿童趣味打字练习 v2\.6\.1

## 专为儿童/英语初学者设计的趣味打字练习网页，纯前端无后端依赖，支持本地直接打开或部署到GitHub Pages。

## ✨ 核心功能

|模块|功能说明|
|---|---|
|🎮 战斗模式|6可选角色/6难度递增怪物，支持打断/连击/暴击/护盾/怒气/道具系统；v2\.6新增4阶段战斗（血量越低攻速伤害越高）、3种随机场景事件（治疗星星/陨石雨/能量球，最高4倍伤害）|

|🔊 双引擎语音|在线TTS（稳定推荐）\+系统TTS双引擎，支持单词/全文朗读、7档语速调节、3档音量、中英/纯英文朗读模式|

## 🚀 快速开始

1. **本地使用**：下载项目到本地后，双击`index.html`即可直接在浏览器打开使用

2. **在线部署**：将项目上传到GitHub仓库，开启Pages服务即可生成在线访问地址

---

## 📖 使用说明

1. 选择练习内容：可自行粘贴文本、上传TXT文件，或直接选择内置教材题库

2. 开始练习：检测到中文内容可选择练习模式，输入时正确字符变绿、错误字符变红，连续正确触发连击特效

3. 战斗模式：通过打字攻击怪物，注意怪物攻击预警可打断，收集道具、触发随机事件击败怪物获得胜利
