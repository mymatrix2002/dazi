# 儿童趣味打字练习 \- README

## 一个专为儿童设计的趣味打字练习网页应用，支持中英文双语练习，带有战斗模式、成就系统和语音朗读功能。纯前端实现，无需后端，可直接在浏览器运行或部署到 GitHub Pages。

## ✨ 功能特性

### 🎮 战斗模式（v2\.4\.0 新增，v2\.6\.0 重大更新）

**角色系统**

- 6 个玩家角色可选：小猫咪、小狗狗、小老虎、小狐狸、小熊熊、小熊猫

- 6 个怪物对手，难度递增：小幽灵（简单）→ 骷髅王（地狱）

- 每个角色有不同属性：血量、攻击力、暴击率、特殊技能
**战斗机制**

- 打对单词 → 玩家攻击怪物，怪物掉血

- 怪物定时自动攻击，注意预警进度条

- 打断系统：预警窗口内连续输入 3 个正确字母可打断攻击

- 连击系统：每 10 连击 \+10% 伤害，每 10 连击获得 1 个护盾

- 暴击系统：不同角色有不同暴击率，暴击双倍伤害

- 护盾系统：护盾可抵挡一次攻击，不扣血

- 怒气系统：受到伤害增加怒气，满 100 可释放怒气技能（3 倍攻击 \+ 回血）

- 道具系统：5 种道具（回血药水、护盾道具、力量药水、无敌星星、怒气药水）

- **阶段化战斗（v2\.6\.0 新增）**：怪物血量分为4个阶段，血量越低攻击越快、伤害越高

    - 普通阶段（100%\~70%血量）：基础攻击速度和伤害

    - 快攻阶段（70%\~40%血量）：攻击速度\+30%，伤害\+20%

    - 狂暴阶段（40%\~20%血量）：攻击速度\+50%，伤害\+50%

    - 绝境阶段（20%以下血量）：攻击速度\+65%，伤害翻倍，预警时间延长

    - 阶段切换有1\.5秒蓄力发光提示，给玩家充足反应时间

- **场景互动机制（v2\.6\.0 新增）**：战斗中随机触发3种场景事件

    - ⭐ 治疗星星：随机位置出现，打完当前句子自动拾取，回复20%最大血量

    - ☄️ 陨石雨：屏幕震动特效\+顶部提示，双方各扣10%血量，护盾/无敌状态可抵消伤害

    - ⚡ 能量球：随机字母A\-Z出现，输入对应字母即可激活，接下来3句伤害翻倍，可与力量药水乘法叠加至4倍伤害

### 🤖 自适应难度AI（v2\.5\.0 新增）

- **智能调节**：根据玩家表现自动调整怪物攻击间隔

- 玩家被击中 → 难度降低（攻击间隔变长）

- 打断成功 → 难度提高（攻击间隔变短）

- 句子完成（准确率 \>90%）→ 小幅提高难度

- **防突然攻击**：调整后至少留 20% 反应时间，避免"突然就被打了"

- **速度限制**：最快不低于基准的 50%，最慢不超过基准的 180%

- **儿童友好**：避免卡关，让每个孩子都能享受战斗乐趣

### 🎯 两大练习模式

- **全文练习模式**：单栏全屏，完整保留中英内容，全文打字练习

- **双语对照模式**：左右分栏，中文对照显示，仅练习英文部分

### 📊 实时统计反馈

- **打字速度**：实时计算每分钟输入单词数（WPM）

- **输入准确率**：双指标统计，准确率 \+ 完成度分开显示

- **练习用时**：精确到秒的计时功能

- **完成进度**：实时显示全文完成百分比

- **进度条可视化**：速度、准确率、进度三条进度条直观展示

### 🔤 逐字符高亮校验

- **正确字符**：绿色高亮 \+ 弹跳动画反馈

- **错误字符**：红色高亮提示

- **当前光标**：闪烁提示，清晰定位

- **待输入字符**：灰色显示，一目了然

### 🎤 双引擎语音系统

**在线语音引擎（推荐）**

- 基于 Cloudflare Worker \+ 百度翻译 TTS 代理

- 解决移动端系统语音加载不出来的问题

- 音质清晰稳定，不受系统语音包限制
**系统语音引擎**

- 本地语音合成，无需网络

- 支持多种系统语音包选择
**语音功能**

- 🔊 全文朗读：逐句播放，高亮同步滚动

- 🔤 单词朗读：单词输入完成自动朗读

- ⚡ 语速调节：0\.25x \- 1\.5x 七档语速

- 🔉 音量调节：轻柔 / 标准 / 响亮三档

- 🎵 朗读模式选择：中英文朗读 / 纯英文朗读

### 🏆 成就激励系统

- **连击贴纸**：5/10/15 连击解锁成就贴纸

- **星星动画**：连击时飘星特效，增加趣味性

- **完成弹窗**：练习完成后详细成绩报告

### 🎨 个性化设置

- **主题切换**：日间 / 暗色双主题，保护视力

- **字号调节**：自由调节练习文本字号

- **音效开关**：按键音、错误音、完成音可独立控制

- **本地持久化**：所有设置自动保存，刷新不丢失

### ⌨️ 虚拟键盘（电脑端）

- **指法引导**：下一个按键自动高亮提示

- **按键反馈**：正确绿色、错误红色，即时反馈

- **大小写切换**：自动同步 Shift 状态

### 📚 内置题库系统

- **教材版本**：广州版（教科版）英语教材，三年级起点

- **覆盖范围**：三到六年级，48 个模块，96 个单元

- **练习类型**：单元单词、重点短语、重点句型、课文对话

- **角色人名**：保留教材原有人名，自动跳过不参与统计

### 📱 移动端深度适配（v2\.6\.1 完善完成）

- 触屏打字优化，滚动流畅不卡顿

- 软键盘弹出三层布局锁定，彻底解决页面上移遮挡头部头像/名称问题

- 所有战斗元素（含v2\.6\.0新增场景特效：治疗星星、陨石雨、能量球）双档位智能缩放，适配不同屏幕高度

- 智能滚动定位，光标始终在可视区域

## 📝 使用说明

### 基本使用流程

1. **选择练习内容**：粘贴内容 / 上传 TXT / 选择题库

2. **开始练习**：点击「开始练习」，检测到中文会弹出模式选择

3. **打字练习**：正确字符变绿，错误字符变红，连续正确触发连击

4. **完成练习**：输完最后一行自动弹出成绩报告

### 战斗模式玩法

1. 点击「⚔️ 战斗模式」进入

2. 选择你的角色（6 种可选）

3. 选择挑战的怪物（6 种难度递增）

4. 通过打字攻击怪物，打对单词造成伤害

5. 注意怪物预警进度条，连续输入 3 个正确字母可打断攻击

6. 收集道具、积累怒气、触发随机场景事件

7. 击败怪物获得胜利！
